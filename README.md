# Syntax-Analyzer
### syntax analyzer for programming language assignment

---

## Util.Token Type

1. LEFT_BRACE = "{"
2. RIGHT_BRACE = "}"
3. CALL = "call'
4. VARIABLE = "variable"
5. IDENT → any names conforming to the C identifier standard
6. PRINT = "print_ari"
7. SEMI_COLON = ";"

## Implementation Method
### Dynamic Scope Language. *(Access Type : Deep Accees)*
